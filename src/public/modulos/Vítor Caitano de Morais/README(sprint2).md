# Escolher os Exercícios da Rotina

O requisito trabalhado nessa sprint deve permitir ao usuário, com base nos exercícios pré-definidos contidos nas estruturas de dados entregues anteriormente, montar uma rotina própria de exercícios para cumprir ao decorrer do dia. 

O requisito deve informar o usuário sobre qual seria a queima calórica de sua rotina, afim de que o usuário possa se manter no controle da queima ou ganho de calorias que vai ter em seu dia, equilibrando as calorias de sua dieta com as calorias aqui queimadas.
